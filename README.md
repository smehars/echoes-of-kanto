# Echoes of Kanto 

**Echoes of Kanto** is a solo hackathon project built for [Mountain Madness 2026](https://csss.ca/) at Simon Fraser University (SFU), hosted by the Computing Science Student Society (CSSS).

You can check out the live project here: **[echoes-of-kanto.vercel.app](https://echoes-of-kanto.vercel.app)**

## Tech Stack

* **Frontend Tooling:** [Vite](https://vitejs.dev/)
* **Framework/Library:** [React / Vue / Vanilla JS] 
* **Deployment:** [Vercel](https://vercel.com/)
* **Other:** ESLint, Node.js
